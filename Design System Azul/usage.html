Usage

Como instalar e usar o Design System Azul no seu projeto.

Instalacao

Copie o arquivo azul.css para seu projeto e inclua no <head>:

<link rel="stylesheet" href="azul.css">

Uso basico

As variaveis CSS estao disponiveis globalmente. Use os tokens diretamente no seu CSS:

.meu-componente {
  background: var(--color-bg-surface);
  border: 1px solid var(--color-border-default);
  color: var(--color-text-primary);
  border-radius: var(--radius-md);
  padding: var(--space-md);
}

.btn-personalizado {
  background: var(--color-brand-primary);
  color: var(--color-text-inverse);
  padding: var(--space-sm) var(--space-md);
  border-radius: var(--radius-md);
}

Dark Mode

O tema escuro e ativado com o atributo data-theme="dark" no elemento <html>:

<html data-theme="dark">

Para criar um toggle, altere o atributo via JavaScript:

document.documentElement.setAttribute('data-theme', 'dark');  // ativar escuro
document.documentElement.setAttribute('data-theme', 'light'); // ativar claro

O estado pode ser salvo no localStorage para persistir a preferencia entre paginas.

Multi-brand

Use data-brand="client-a" (ou b, c) para mudar a paleta brand sem alterar o restante:

<html data-brand="client-a">
<!-- Todas as ocorrencias de --color-brand-primary serao azul -->

Variantes disponiveis:

  • data-brand="client-a" — Blue (#2563EB)
  • data-brand="client-b" — Amber (#F59E0B)
  • data-brand="client-c" — Purple (#7C3AED)

Alto Contraste

Adicione data-contrast="high" para aumentar o contraste de texto e bordas:

<html data-contrast="high">

Exemplo completo

<!DOCTYPE html>
<html lang="pt-BR" data-theme="dark">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Meu Projeto</title>
  <link rel="stylesheet" href="azul.css">
</head>
<body>
  <nav style="background:var(--color-bg-elevated);padding:16px 24px;
             border-bottom:1px solid var(--color-border-default)">
    <strong>Meu App</strong>
  </nav>
  <main style="padding:32px;max-width:800px;margin:0 auto">
    <h1 class="text-2xl">Bem-vindo!</h1>
    <p class="text-secondary">Este e um projeto usando o Design System Azul.</p>
    <button class="btn btn--primary">Comecar</button>
  </main>
</body>
</html>
